# Construction of a Personal Computer System

A **Personal Computer (PC) system** is an electronic device made up of hardware and software components that work together to process data, store information, and perform tasks for users.

---

# Main Components of a Personal Computer System

## 1. System Unit

The **system unit** is the main casing that contains the internal components of the computer.

### Components inside the System Unit
- Motherboard
- Processor (CPU)
- RAM
- Storage devices
- Power Supply Unit (PSU)
- Cooling system
- Expansion cards

---

# 2. Motherboard

The **motherboard** is the main circuit board that connects all components together.

## Functions of the Motherboard
- Connects hardware components
- Allows communication between devices
- Supplies power to components
- Contains processor socket and RAM slots

## Main Parts of a Motherboard
- CPU socket
- RAM slots
- Chipset
- SATA ports
- M.2 slots
- PCIe slots
- BIOS/UEFI chip

---

# 3. Central Processing Unit (CPU)

The **CPU** is the brain of the computer. It performs calculations and controls system operations.

## CPU Functions
- Executes instructions
- Processes data
- Controls hardware communication

---

# Processor Sockets

A **processor socket** is the connector on the motherboard where the CPU is installed.

Different CPU generations require different sockets because of changes in:
- Power delivery
- Pin layout
- Memory support
- Chipset technology

---

# Intel Processor Sockets

## Early Intel Sockets

| Generation/Period | Socket | Description |
|---|---|---|
| 1980s | Socket 1 | Early Intel 486 processors |
| 1990s | Socket 3 | Intel 486 DX processors |
| Mid-1990s | Socket 5 | First Pentium CPUs |
| Late 1990s | Socket 7 | Pentium MMX processors |

---

## Pentium II and III Era

| Socket/Slot | Description |
|---|---|
| Slot 1 | Cartridge-based Pentium II |
| Socket 370 | Pentium III and Celeron |

---

## Pentium 4 Era

| Socket | Description |
|---|---|
| Socket 423 | First Pentium 4 socket |
| Socket 478 | Improved Pentium 4 support |
| LGA 775 | First major Intel LGA socket |

---

## Intel Core Series Sockets

| Socket | Supported Generations |
|---|---|
| LGA 1156 | 1st Gen Intel Core |
| LGA 1155 | 2nd & 3rd Gen |
| LGA 1150 | 4th & 5th Gen |
| LGA 1151 | 6th to 9th Gen |
| LGA 1200 | 10th & 11th Gen |
| LGA 1700 | 12th–14th Gen |
| LGA 1851 | Current/Newer platforms |

---

# Intel LGA Socket Design

Intel mainly uses **LGA (Land Grid Array)** sockets.

## Features
- Pins are on the motherboard
- CPU has contact pads

## Advantages
- Supports more pins
- Better electrical performance
- Stronger CPU durability

## Disadvantages
- Motherboard pins bend easily
- Motherboard replacement may be expensive

---

# AMD Processor Sockets

AMD historically used **PGA (Pin Grid Array)** sockets.

## PGA Features
- Pins are on the CPU
- Socket contains holes

---

## Early AMD Sockets

| Socket | Description |
|---|---|
| Socket A (462) | Athlon and Duron |
| Socket 754 | Early Athlon 64 |
| Socket 939 | Gaming systems |
| Socket AM2 | DDR2 support |
| Socket AM2+ | Improved HyperTransport |
| Socket AM3 | DDR3 support |
| Socket AM3+ | FX processors |

---

## Modern AMD Sockets

| Socket | Description |
|---|---|
| FM1/FM2 | AMD APUs |
| AM4 | Ryzen 1000–5000 |
| AM5 | Ryzen 7000 and newer |

---

# AMD Socket Types

## AM4 Features
- DDR4 support
- PCIe Gen 4
- Long-term compatibility

### Advantages
- Excellent upgrade path
- Affordable motherboards
- Large ecosystem

### Disadvantages
- Limited to DDR4
- Older architecture

---

## AM5 Features
- DDR5 support only
- PCIe Gen 5
- LGA-based design

### Advantages
- Future-ready
- Faster bandwidth
- Better power delivery

### Disadvantages
- Higher cost
- DDR5 RAM is expensive

---

# 4. RAM (Random Access Memory)

RAM temporarily stores data currently being used by the CPU.

## Functions of RAM
- Temporary data storage
- Improves multitasking
- Speeds up applications

---

# RAM Slots

RAM modules are inserted into slots on the motherboard.

| Slot Type | Used In |
|---|---|
| DIMM | Desktop PCs |
| SO-DIMM | Laptops |

---

# RAM Generations

## DDR RAM Generations

| Generation | Speed Range | Voltage |
|---|---|---|
| DDR | 200–400 MT/s | 2.5V |
| DDR2 | 400–1066 MT/s | 1.8V |
| DDR3 | 800–2133 MT/s | 1.5V |
| DDR4 | 1600–5333 MT/s | 1.2V |
| DDR5 | 4800+ MT/s | 1.1V |

---

# RAM Characteristics

## 1. Capacity

Common RAM sizes:
- 4GB
- 8GB
- 16GB
- 32GB
- 64GB+

### Advantages of Larger RAM
- Better multitasking
- Improved gaming
- Faster editing/rendering

### Disadvantages
- Expensive
- Unused RAM gives no extra benefit

---

## 2. Frequency (Speed)

Measured in:
- MHz
- MT/s

### Advantages
- Faster data transfer
- Better system performance

### Disadvantages
- Generates more heat
- Higher price

---

## 3. Memory Channels

| Type | Description |
|---|---|
| Single Channel | One RAM stick |
| Dual Channel | Two matching sticks |
| Quad Channel | Four matching sticks |

### Advantages
- Increased memory bandwidth
- Better CPU performance

### Disadvantages
- Requires matching RAM modules

---

## 4. ECC vs Non-ECC RAM

| Type | Purpose |
|---|---|
| ECC RAM | Detects/corrects memory errors |
| Non-ECC RAM | Standard consumer RAM |

### ECC Advantages
- More reliable
- Used in servers/workstations

### ECC Disadvantages
- More expensive
- Slightly slower

---

# 5. Storage Devices

Storage devices permanently store data and programs.

---

# Types of Storage Devices

## 1. Hard Disk Drive (HDD)

Uses spinning magnetic disks.

### Advantages
- Cheap per GB
- Very large capacity
- Good for backups

### Disadvantages
- Slow speed
- Mechanical failure risk
- Noisy
- Higher power usage

---

# 2. Solid State Drive (SSD)

Uses flash memory and contains no moving parts.

---

# Types of SSDs

## A. SATA SSD

Uses SATA interface.

### Features
- Similar shape to HDD
- Speeds up to ~600 MB/s

### Advantages
- Faster than HDD
- Affordable
- Silent operation

### Disadvantages
- Slower than NVMe SSD

---

## B. mSATA SSD

Mini SATA SSD used in older laptops.

### Advantages
- Compact size
- Better than HDD

### Disadvantages
- Becoming obsolete
- Limited speed

---

## C. M.2 SATA SSD

Uses M.2 slot but SATA protocol.

### Advantages
- Compact
- No cables required

### Disadvantages
- SATA speed limitations

---

## D. NVMe SSD

Uses PCIe interface for very high speed.

### Features
- Much faster than SATA
- Installed in M.2 slots

### Advantages
- Extremely fast
- Low latency
- Excellent for gaming and editing

### Disadvantages
- Expensive
- Generates heat

---

## E. PCIe Add-in Card SSD

SSD mounted on PCIe expansion card.

### Advantages
- Very high performance
- Better cooling

### Disadvantages
- Expensive
- Uses PCIe slot space

---

## SSD Form Factors

| Form Factor | Description |
|---|---|
| 2.5-inch | Traditional SSD shape |
| M.2 | Thin compact SSD |
| U.2 | Enterprise SSD |
| PCIe Card | Expansion card SSD |

---

# SSD Advantages

- Very fast
- Silent
- Lower power consumption
- Durable
- Lightweight

---

# SSD Disadvantages

- More expensive than HDD
- Limited write lifespan
- Smaller capacities may cost more

---

# 3. Hybrid Drive (SSHD)

Combines HDD and SSD technologies.

### Advantages
- Faster than HDD
- Larger capacity
- Lower cost than SSD

### Disadvantages
- Slower than SSD
- More complex design

---

# 4. Optical Storage

Examples:
- CD
- DVD
- Blu-ray

### Advantages
- Cheap media
- Good for archival

### Disadvantages
- Slow
- Limited capacity
- Becoming obsolete

---

# 5. USB Flash Drives

Portable flash storage devices.

### Advantages
- Portable
- Easy file transfer
- Durable

### Disadvantages
- Easy to lose
- Limited lifespan

---

# 6. Cloud Storage

Stores data on internet servers.

### Advantages
- Accessible anywhere
- Automatic backups
- Easy sharing

### Disadvantages
- Requires internet
- Privacy concerns
- Subscription costs

---

# Storage Comparison Table

| Storage Type | Speed | Capacity | Cost | Reliability |
|---|---|---|---|---|
| HDD | Slow | Very High | Cheap | Moderate |
| SATA SSD | Fast | High | Moderate | High |
| NVMe SSD | Very Fast | High | Expensive | High |
| SSHD | Moderate | High | Moderate | Moderate |
| Optical | Very Slow | Low | Cheap | Moderate |
| USB Flash | Moderate | Moderate | Moderate | Moderate |
| Cloud Storage | Internet Dependent | Very High | Subscription | High |

---

# 6. Power Supply Unit (PSU)

The PSU converts AC electricity into DC power for computer components.

## Functions
- Supplies stable power
- Protects hardware from power issues

---

# 7. Cooling System

Cooling systems prevent overheating.

## Types
- Air cooling
- Liquid cooling

### Cooling Components
- Fans
- Heat sinks
- Thermal paste

---

# 8. Input Devices

Used to enter data into the computer.

## Examples
- Keyboard
- Mouse
- Scanner
- Microphone

---

# 9. Output Devices

Display processed information.

## Examples
- Monitor
- Printer
- Speakers

---

# 10. Expansion Cards

Add extra functions to the computer.

## Examples
- Graphics card (GPU)
- Sound card
- Network card

---

# Construction Steps of a Personal Computer

1. Install CPU into motherboard socket
2. Apply thermal paste and cooler
3. Insert RAM into slots
4. Mount motherboard into case
5. Install storage drives
6. Connect power supply
7. Install expansion cards
8. Connect cables
9. Attach peripherals
10. Install operating system

---

# Conclusion

A personal computer system is built from several hardware components working together to process, store, and display information.

Modern systems commonly use:
- Intel LGA 1700/LGA 1851 sockets
- AMD AM5 sockets
- DDR5 RAM
- NVMe SSD storage

Advancements in processor sockets, RAM technology, and storage devices have greatly improved computer performance, reliability, and efficiency.
